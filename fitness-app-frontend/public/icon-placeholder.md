# Iconos PWA

Para completar la PWA, necesitas crear los siguientes iconos:

## Iconos Requeridos

1. **icon-192.png** - 192x192 píxeles
2. **icon-512.png** - 512x512 píxeles

## Herramientas Recomendadas

Puedes usar estas herramientas para generar los iconos:

1. **RealFaviconGenerator**: https://realfavicongenerator.net/
   - Sube una imagen y genera todos los tamaños necesarios

2. **PWA Builder Image Generator**: https://www.pwabuilder.com/imageGenerator
   - Genera iconos optimizados para PWA

3. **Favicon.io**: https://favicon.io/
   - Genera iconos desde texto o imagen

## Diseño Sugerido

- **Color primario**: #FF6D1F (naranja)
- **Color secundario**: #86c4c2 (turquesa)
- **Fondo**: Puede ser sólido o con gradiente
- **Icono**: Puede ser una "F" estilizada, un logo de fitness, o un ícono de pesas

## Ubicación

Coloca los archivos generados en:
- `fitness-app-frontend/public/icon-192.png`
- `fitness-app-frontend/public/icon-512.png`

## Verificación

Una vez creados, la PWA estará completamente funcional y podrá ser instalada en dispositivos móviles.
